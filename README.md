# OpenAPITemplate.jl

A [PkgTemplates.jl](https://github.com/JuliaCI/PkgTemplates.jl) plugin that
scaffolds Julia REST/JSON API wrapper packages.

The generator produces a two-layer architecture:

- `src/api/` — generated by [OpenAPI Generator](https://openapi-generator.tech/)
  (`julia-client`), committed as plain Julia, never re-run at end-user runtime.
- `src/client/` — hand-written ergonomic overlay providing a `Client` struct,
  middleware, auth strategies, pagination iterators, and pretty printing.

See [`IMPLEMENTATION_PLAN.md`](IMPLEMENTATION_PLAN.md) for the full design and
phased rollout.

## Quick start

```julia
using PkgTemplates, OpenAPITemplate

t = Template(;
    user = "your-username",
    plugins = APIWrapper(;
        spec_url = "https://petstore3.swagger.io/api/v3/openapi.json",
    ),
)
t("PetstoreClient")
```
